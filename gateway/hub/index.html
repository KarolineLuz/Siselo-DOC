Homologacao Local

SISELO em uma porta so

Esta stack centraliza o ambiente local em uma unica porta. A aplicacao principal roda na raiz, e os demais repositorios ficam acessiveis por rotas dedicadas para navegacao e homologacao.

Aplicacao Principal `/` aponta para o Siselo-Users com banco MySQL compartilhado. / Repositorio de Documentacao Navegue pelo Siselo-DOC sem abrir outra porta. /docs/ Repositorio Siselo-Users Acesse arquivos e estrutura do backend rapidamente. /users-repo/ Repositorio Siselo-Admin Exposto para homologacao enquanto o modulo ainda nao tem app propria. /admin-site/ Repositorio Siselo-Frontend Disponivel pela mesma porta enquanto a interface ainda evolui. /frontend-site/ Navegador Geral de Repositorios Lista completa do workspace com todos os repositorios do Siselo. /repos/

Quando `Siselo-Admin` e `Siselo-Frontend` virarem apps executaveis, basta trocar os destinos dessas rotas no gateway sem mudar o ponto de entrada da homologacao.
